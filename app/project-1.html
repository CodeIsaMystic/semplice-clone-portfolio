"Git" A Workflow

DBSH  template est un projet d'étude, travaillé avec un client potentiel, un simple site vitrine servant de portfolio, de carte de visite pour coach sportif. Le Landing Page Optimization a été étudié pour une simple fonctionnalité: une fenêtre ouverte sur le Web.

Objectifs

Prendre en charge un projet utilisant les méthodes, outils et concepts professionnels.

Stack

HTML5, CSS3, Vanilla Javascript

NPM, Git, Github

WebPack, Babel, PostCSS, Autoprefixer

Topics

BEM, Atomic Design

Files Architecture, Automation

SEO, Accessibility
